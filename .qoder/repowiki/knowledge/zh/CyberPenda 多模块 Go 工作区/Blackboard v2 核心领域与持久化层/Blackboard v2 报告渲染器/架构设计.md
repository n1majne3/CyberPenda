本模块是纯渲染层，不持有状态、不访问持久化，仅依赖 `pentest/internal/blackboardv2` 的投影类型。
- 接口契约：通过 `V2Reader` / `CTFV2Reader` 两个小接口注入数据源，由调用方提供实现；`V2Request` 仅携带 `ProjectID`，屏蔽底层存储细节。
- 生成器：`V2Generator` 与 `CTFV2Generator` 分别对应 Pentest 报告与 CTF Solution 两种输出路径，均通过 `Generate(ctx, request)` 返回统一的 `Report`（status/format/markdown）。
- 渲染管线：`RenderV2Markdown` / `RenderCTFV2Markdown` 为对外导出函数，内部委托给 `renderV2Markdown` / `renderCTFV2Markdown`，后者再按段落/列表/证据等小节拆分为 `writeV2*` 辅助函数，形成“入口 → 结构编排 → 片段写入”的单向调用链。
- 文本安全：所有用户可控字段在单行场景下经 `escapeV2Markdown` 转义 Markdown 元字符，多行内容走 `writeV2LiteralBlock` 以 4 空格缩进呈现为代码块，避免注入。
- 产物约定：`Report` 结构体是模块唯一对外数据结构，`Format: "markdown"` 固定值表明当前仅支持 Markdown 输出。

Two concerns live in the `daemon` package under this scope:

- Provider session lifecycle (`provider_session_control.go`, `provider_session_factory.go`). A `providerSessionRegistry` is a daemon-owned, in-memory `map[string]runtime.ProviderSession` keyed by task ID; it is intentionally volatile so restarts fail closed rather than reattach orphaned stdio processes. `BindProviderSession` attaches a provider-native session to an existing Task, installs an optional `ProviderSessionEventSink` callback, and funnels unsolicited provider notifications through `persistProviderSessionEvent`, which whitelists a fixed set of payload keys (e.g. `provider`, `session_id`, `outcome`) before appending them to the active Continuation or the Task event log — raw protocol frames never leak into the conversation. `closeTask`/`closeAll` drive idempotent shutdown, ignoring `ErrProviderSessionClosed`. Native steering helpers (`nativeSteerMode`, `nativeSteerState`, `nativeSteerOperation`) map capability flags to one of two modes (`InTurnSteer`, `InterruptThenReplace`) and dispatch the right method on the session.

- Launch assembly seam (`provider_session_factory.go`). `ProviderSessionFactory` / `ProviderSessionBinding` are the boundary between the daemon and concrete provider implementations: launch requests carry only Task/Continuation/Runner/provider metadata, while credentials and raw transport frames stay inside the adapter. `supportsPersistentProviderSession` gates whether the runner/provider pair uses the persistent path (Codex, Claude Code, Pi) versus the legacy one-shot path. Errors are wrapped in `providerSessionFactoryError` with a stable user-facing message but an unwrap chain for diagnostics.

- Auth gate (`auth_test.go` exercises the behavior). The server rejects non-loopback binds when no `AuthToken` is configured, accepts loopback without a token, requires a `Bearer <token>` header on `/api/*` routes, allows `/health`, SPA entry/static assets, and the sandbox MCP transport via a `?token=` query param, and still requires a token for non-GET methods hitting asset paths.
